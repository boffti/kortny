# Tone of voice guide

This reference defines the default voice spectrum. Adapt to your workspace brand using known workspace facts where available.

## Voice dimensions

Every brand voice sits somewhere on these five axes:

| Axis | Range |
|---|---|
| Formality | Formal ←→ Casual |
| Warmth | Clinical ←→ Warm |
| Confidence | Deferential ←→ Assertive |
| Complexity | Plain ←→ Technical |
| Humor | Serious ←→ Playful |

## Default professional voice (baseline)

Used when no brand facts are present:
- **Formality**: slightly casual (contractions OK; no slang)
- **Warmth**: warm but not gushing
- **Confidence**: assertive; take a position
- **Complexity**: plain language; explain jargon when used
- **Humor**: light; do not force it

## Common voice archetypes

### Trustworthy authority

Formality: formal / Warmth: measured / Confidence: very assertive / Complexity: plain
- Lead with the conclusion.
- Use active voice. Avoid hedging language ("we believe", "it seems").
- Numbers and specifics over vague claims.
- No exclamation marks.

### Friendly expert

Formality: casual / Warmth: warm / Confidence: assertive / Complexity: plain-to-moderate
- Use "you" and "we" freely.
- Contractions throughout.
- One moment of lightness per message is acceptable; more is too much.
- Technical terms explained inline.

### Premium / aspirational

Formality: formal / Warmth: cool / Confidence: assertive / Complexity: moderate
- Short sentences. White space.
- Remove redundant adjectives.
- No exclamation marks. No emoji.
- Quality signals: precision, restraint.

## Vocabulary guidance

When adapting content to a brand voice, check these patterns:

**Remove** (unless brand calls for them):
- Filler openers: "I'm excited to share…", "We're thrilled to announce…"
- Corporate hedge: "leverage", "synergy", "going forward", "at the end of the day"
- False intimacy: "team" when addressing external customers
- Overuse of "very", "really", "truly"

**Prefer**:
- Specific nouns over generic ones ("the checkout flow" not "the experience")
- Active verbs ("we shipped" not "shipping has been completed")
- Direct requests ("reply by Friday" not "please do not hesitate to respond")
